# Personal website

No fireworks (*yet!*), no 404s either (*hopefully!*).

## Setup

Install the dependencies:

```
yarn install
```

Build: 

```
npm run build
```

Setup automatic build:

```
npm run watch
```

Run webpack-dev-server:

```
npm run dev
```

Run express server:

```
npm run start
```

## Stack
- React
- Sass
- Webpack
- Babel
- Yarn
- Express
- Heroku
